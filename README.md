# Codex Skills

Персональные скилы для проектирования, разработки, тестирования и документирования
Python-сервисов с DDD, CQRS и гексагональной архитектурой.

## Установка

Скопируйте нужный каталог в глобальную директорию скилов Codex:

```bash
cp -R <skill-name> ~/.codex/skills/
```

Каждый каталог является самостоятельным скилом и содержит файл `SKILL.md`.

## Состав

- документирование бизнес-контекстов и CQRS-пайплайнов;
- проектирование domain- и application-слоёв;
- Postgres-репозитории и конфигурация;
- FastAPI-, NATS- и subscription-воркеры;
- pytest-тесты, GitLab CI и документация репозитория.
